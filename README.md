# 📚 Curso PROA - Desenvolvimento Full Stack

Este repositório contém meus estudos, projetos e exercícios desenvolvidos durante o curso **PROA**, com duração de 6 meses.

## 🚀 Sobre o curso

O **PROA** é um programa de formação que tem como objetivo capacitar jovens na área de tecnologia, preparando para o mercado de trabalho como desenvolvedor.

Durante o curso, estou aprendendo desde o básico até conceitos mais avançados de desenvolvimento web e programação.

## 🧠 Tecnologias e conhecimentos adquiridos

Ao longo da formação, estou estudando:

* 🌐 HTML (estruturação de páginas)
* 🎨 CSS (estilização e responsividade)
* ⚡ JavaScript (interatividade e lógica)
* ☕ Java (programação orientada a objetos)
* 🗄️ Banco de Dados (modelagem e consultas SQL)
* 🎨 Figma (design de interfaces e prototipagem)
